Projet ISN - Charly John Canals



Cette page contient mes premières idées de projet à mener cette année.
Le projet sera uniquement logiciel dans l'idéal.

Orienté "Data science", création et gestion de base de données dans un cas pratique.
Il pourra être mis en oeuvre les bibliothèques MatPlotLib, NumPy pour mettre en place des visuels/graphes des données mises en jeu.

Ce pourrait être également l'occasion d'apprendre et de mettre en oeuvre la librairie connue sous le nom de Pandas, contenant des outils divers pour manipuler des données efficacement.





    Caractéristiques du projet :
  • Basé sur une situation réaliste, un besoin à déterminer que ce soit gérer une base de données (météo, entreprise commerciale, application financière basique)
  • Dispose d'une interface simple, programme fenêtré avec potentielle UI basique (Tkinter ou autre)


      • Idées de projets possibles :
    • Application de visualisation boursière, avec outils de suivi, différents modes de présentation des données, cours des devises, etc.
    • Registre informatisé pour application particulière (hôtel, système de suivi d'inventaire ou autre)


    J'aimerais également apprendre à créer et manipuler des flux RSS, utiles à la création d'un système d'alertes, ou encore m´intéresser au développement d'une web application (Django/Flask).
